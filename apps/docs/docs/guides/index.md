## Contributing

If you'd like to add a new guide, please follow these steps:

1. Create a new Markdown file in `apps/docs/docs/guides/`
2. Add the guide to the sidebar in `apps/docs/docs/.vitepress/config.ts`
3. Submit a pull request

## Questions?

If you have questions or need help, please open an issue on GitHub.
